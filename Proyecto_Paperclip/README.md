# Operación Paperclip y el Dilema Ético de la Carrera Espacial

Una página web educativa interactiva diseñada con un enfoque riguroso, crítico y humanista. Explora el desarrollo técnico del cohete V2, el reclutamiento secreto de científicos alemanes por parte del gobierno estadounidense tras la Segunda Guerra Mundial y el trabajo esclavo en el campo de concentración de Mittelbau-Dora que hizo posible la llegada a la Luna.

## 🚀 Características Interactivas
- **Modo Expediente Clasificado:** Alterna entre un diseño moderno oscuro y una interfaz retro inspirada en dossiers desclasificados de inteligencia.
- **Censura Interactiva:** Haz clic en los bloques negros de censura para revelar información confidencial.
- **Explorador de Fuente Primaria:** Pestañas dinámicas para analizar testimonios de sobrevivientes.
- **Línea de Tiempo Dinámica:** Hitos clave expandibles desde 1942 hasta 1969.
- **Calibrador Ético:** Control deslizante interactivo que analiza en tiempo real tu postura sobre el pragmatismo tecnológico y la ética absoluta.

## 🌐 Publicar en GitHub Pages
Para subir este sitio a GitHub Pages y solucionar el error `404 File not found`:
1. Asegúrate de incluir el archivo `index.html` en la raíz de tu repositorio (este archivo ya ha sido creado).
2. Sube todos los archivos de esta carpeta a tu repositorio de GitHub.
3. Ve a **Settings (Configuración) > Pages** en tu repositorio de GitHub.
4. En **Build and deployment**, selecciona la rama `main` (o `master`) y la carpeta `/root`, luego presiona **Save (Guardar)**.
5. Tu sitio estará disponible públicamente en pocos minutos.
